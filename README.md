# Real-Time Systems

Repository of the subject **Real-Time Systems** (final year of Computer Engineering), organized by practical guides.

## Repository Structure

- `Guide 1/`: Introductory exercises in C (operations, strings, sorting, semaphore and stopwatch).
- `Guide 2/`: exercises focusing on GPIO and real-time events (Raspberry Pi + `pigpio`).

## Documentation by folder

Cada carpeta incluye su **README propio** con el detalle de:

- Objectives of the exercises,
- Requirements,
- Compilation and execution,
- Technical notes.

You can check:

- [Guide 1/README.md](Guide%201/README.md)
- [Guide 2/README.md](Guide%202/README.md)

## General Objective

Practice real-time systems concepts through progressive exercises, from basic C logic to hardware handling, interrupts, and latency/jitter measurements.
